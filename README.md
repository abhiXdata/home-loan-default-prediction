# 🏠 Home Loan Default Prediction System

An end-to-end machine learning application for predicting home loan default risk using advanced ML techniques.

## 📊 Project Overview

This system uses machine learning to predict the likelihood of home loan default by analyzing multiple data sources including client demographics, credit history, and financial indicators. The model provides accurate risk assessments to support informed lending decisions.

## ✨ Features

- **📊 Interactive Dashboard**: Real-time KPIs and project overview
- **📈 EDA Visualizations**: Comprehensive exploratory data analysis
- **🧹 Preprocessing Pipeline**: Automated data cleaning and feature engineering
- **🤖 Model Comparison**: Multiple ML models with performance metrics
- **🏦 Credit Risk Prediction**: Real-time applicant risk assessment
- **📱 Responsive Design**: Optimized for all devices

## 🛠️ Tech Stack

- **Frontend**: Streamlit, Plotly
- **Backend**: Python, Pandas, NumPy
- **ML Models**: Scikit-Learn (Decision Tree, Random Forest, Logistic Regression, Naive Bayes)
- **Data Processing**: SMOTE, One-Hot Encoding, Feature Engineering
- **Deployment**: Streamlit Cloud / Docker
